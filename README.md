## Installation
Windows, macOS, Linux 사용자들은 본 리포지토리의 [릴리즈 페이지](https://github.com/KimuSoft/muvel-public/releases)로 이동하여 운영체제에 맞는 빌드를 다운받아주세요.
Android 사용자들은 [Play Store](https://play.google.com/store/apps/details?id=com.muvel&hl=ko), iOS는 홈페이지의 베타 링크를 사용해주세요.

* Latest LTS: https://github.com/KimuSoft/muvel-public/releases/latest
